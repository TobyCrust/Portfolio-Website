Archery Simulator An Archery simulator made with Unreal Engine. Complete with an escape level.

Project Overview

I have created an immersive archery simulator inside of unreal engine. This project taught me alot about using the novelties of VR to their full advantage. Being able to draw back the string and shoot around corners made it feel so much more immersive than the point and click experiences that have come to VR.

This project was featured in the Victoria University end of year design exhabition.

Tools Used

Unreal Engine
Blender
Oculus Quest 2
